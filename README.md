# Welcome to Book Store 

## You will find in this repo the following stuff:
* REST API for the Book Store
* REST clients should be able to 
    * Get a list of books
    * Get a single book by id
    * Add a new book
    * Update a book
    * Delete a book
* Web UI for the Book Store
* Users should be able to 
    * Get a list of books
    * Add a new book
    * Update a book
    * Delete a book
